# Web_beadando_MateKristof_ACXIFQ

## Capabilities:
CRUD operations for the Products table using the Admin page

## How to Build:
1. ```
   cd db/
   docker-compose up --build
   ```
2. ```
   cd backend/
   mvn spring-boot:run
   ```
3. ```
   cd frontend/
   npm start
   ```
4. Head to the http://localhost:3000/
